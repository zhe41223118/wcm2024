# Replit key建立
講解Replit分組平台倉儲設定。(詳細觀看<a href="https://www.youtube.com/watch?v=syTeJ0kJAow">此YT</a>
)

主要內容有用.ssh-keygen創建自己的config和id_rsa，刪除重新綁定在replit的倉儲帳號以利重新獲得自己的權限。
