# 如何將影片嵌入進自己網站

將自己想上傳的影片內容

先放到yt上，之後至自己的yt影片下

點擊分享

選擇第一個嵌入

將顯示在右側的超連結程式碼複製

來到動態編輯網站

點擊Source code(最下排笑臉右邊)

將複製的超連結內容貼上

完成。

(註：如需有特別需要可以修改超連結內容來完成所需

例：width="560" height="315" 為影片的長寬設定)
