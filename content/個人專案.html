個人專案

我想要製作一個俄羅斯方塊。

現在的初版設計為將這個網站的資料導入給GPT，然後請他幫我製作一個遊戲的初版代碼。

v1.7z (初版遊戲)

內容主要為：

.html (講述了網頁上各元素的意義和屬性)

.css (用來幫結構化文件.html 添加樣式的電腦語言)

.js (一門支援物件導向程式設計、指令式編程和函數式程式設計)

三個程式代碼執行，現在這個遊戲是能夠遊玩，但缺點也是很多：

1.網頁只能以資料夾所在位置開啟，不能在網址上開啟。

2.遊戲本身過於單調 方塊翻轉問題缺陷。

經過以上幾點問題，我重新將程式整理好

V2.7z (修改後第二版遊戲)

新增了：

1.遊戲最高分紀錄

2.遊戲開始前玩家名稱輸入

3.按下空白鍵能夠暫停遊戲

但問題也是有的，方塊一樣有修改，卻變得無法轉動，按暫停時遊戲直接結算歸零，遊戲玩家名稱輸入視窗點擊會不斷跳出等。

PS：

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('GAME.html')

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8000, debug=True)

此為在詢問claude問得的設置遊戲網址的程式代碼(須是.py形式)
